# hostel-management-
hostel management project <br>
A simple web-based Hostel Management System developed using PHP and MySQL.
This project helps manage student details, room allocation, and booking information efficiently.<br>
🚀 Features<br>
👤 Student Registration & Login<br>
🏠 Room Selection<br>
📋 Personal Details Management<br>
💳 Payment Handling<br>
📄 Booking Status Tracking<br>
🔐 Session-based Authentication<br>
🛠️ Technologies Used<br>
Frontend: HTML, CSS<br>
Backend: PHP<br>
Database: MySQL<br>
Server: XAMPP / WAMP<br>
