# hostel-management-
hostel management project <br>
A simple web-based Hostel Management System developed using PHP and MySQL.
This project helps manage student details, room allocation, and booking information efficiently.<br>
🚀 Features
👤 Student Registration & Login
🏠 Room Selection
📋 Personal Details Management
💳 Payment Handling
📄 Booking Status Tracking
🔐 Session-based Authentication
🛠️ Technologies Used
Frontend: HTML, CSS
Backend: PHP
Database: MySQL
Server: XAMPP / WAMP
